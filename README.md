# ✅ Ma Todo List 

Un espace web interactive pour organiser ses tâches quotidiennes. Ce projet démontre ma capacité à gérer l'interaction utilisateur et le stockage de données côté client.

### 🚀 Fonctionnalités
- **Ajout rapide** : Création de tâches en un clic.
- **Persistance** : Les tâches restent affichées même après avoir actualisé la page (LocalStorage).
- **Gestion d'état** : Possibilité de barrer les tâches terminées ou de les supprimer.
- **Design Responsif** : Interface moderne et épurée (Neumorphism).

### 🛠️ Technologies utilisées
- HTML5 / CSS3
- JavaScript (Manipulation du DOM)

- LocalStorage API
